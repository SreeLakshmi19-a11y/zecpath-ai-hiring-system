# Zecpath AI System
## Overview
Zecpath AI is an intelligent hiring system that automates recruitment using
AI.
## Modules
- Resume Parser
- ATS Scoring Engine
- Screening AI
- Interview AI
- Decision Engine
## Setup Instructions
1. Clone repository
2. Create virtual environment
3. Install dependencies:
pip install -r requirements.txt
4. Run:
python main.py
## Folder Structure
- data/: Input datasets
- parsers/: Resume & JD parsing
- ats_engine/: Scoring logic
- screening_ai/: Screening logic
- interview_ai/: Interview logic
- scoring/: Final decision logic
- utils/: Helper functions
- tests/: Test scripts
## Logging
Logs are stored in `ai_logs.log`
## Testing
Run tests using:
pytest tests/
